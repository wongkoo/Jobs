## Description

  **应届生们，面对各种公司各种职位，各种官网账号，不同的邮箱，当大家海投的时候，是否感到混乱难以梳理？**

  **Jobs来了！哪个公司，哪个职位，面试笔试等提醒，轻松应对hr的突然来电，不再忘记当初自己报了哪个职位。选填官网账号、报名邮箱、密码提示，轻松存储本地。**

  **Jobs，让求职清醒而高效。**
